What Makes It Different

🔗

Hash-Chain Provenance

Every memory block is cryptographically linked to the previous. Tamper any block — all subsequent hashes break. Full audit trail, no trust required.

Cryptographic
🧠

Temporal Decay

Memory blocks cool down over time like human neurons. Recently accessed knowledge stays hot, old unused facts fade. Never deleted, just forgotten.

Novel
📊

Drift Detection

Monitors register distribution across sessions. Catches when agent starts expressing opinions as facts or losing confidence. Cross-session anomaly alerts.

Unique
🏛️

Provenance Archaeology

Trace any decision back to its root facts. "Why did we decide X?" → full reasoning chain reconstructed from the hash chain.

Debugging
🤖

Auto-Triggering Agents

sint-main runs automatically as primary agent. Plugin hooks auto-log findings. MCP server provides 16 memory tools. No manual invocation needed.

Automatic
🎯

Semantic Registers

Every output typed: SENSE (raw), FACT (verified), LOGIC (inference), OPINION (subjective), ACTION (plan). Mixing without labeling = failure.

SINT-UA

Semantic Registers

Register Definition Requirement
SENSE Raw data, observation, user input No interpretation added
FACT Verifiable claim about the world Source required
LOGIC Deductive inference from FACT/SENSE Premises must be explicit
OPINION Subjective assessment, preference Actor must be declared
ACTION Action plan, execution chain Each step must follow from prior
